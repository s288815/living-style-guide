What is Living Style Guide?

Is a place to store and document all the code used for the website. The documentation contain detailed CSS information about each section of the website. It can transform front end code into well described pattern libraries with minimal efforts. It is also good platform to connect designer and developer to reference and discuss changes to the site.


Why is it useful?

As mentioned above it has the ability to connect both frontend developer and backend developer together. Hence with better communication project can be set in a faster pace. next, Living style guide can avoid CSS duplication and standardize CSS, making both designers and developers can make changes easily to the existing CSS for a new design hence keeping the codebase small. It is also useful when a new team member join the team, they can easily catch up with existing member as they can refer to living style guide.


List of Living Style Guide tips

  • Develop living style guide at the very begining of the project. (Robertson, 2014)
  • Always start out with basic things color palette, grid layout, basic styles for headers and body text. (Robertson, 2014)
  • Having code standards document will make coding easier to manipulate and easier to be maintained. (Debenham, 2011)
